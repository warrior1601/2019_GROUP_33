Detailed Description

A 3x3 matrix with private members of a single floating-point 2D array.

This class has numerous operator overloads for both scalars and other matrix3x3 objects; abilities to initialise the matrix3x3 as a rotation matrix for cells; ability to transpose the matrix3x3; ability to determine the determinant; and ability to get the inverse of a matrix3x3.

void Matrix3x3::Initialise_As_Rotation_Matrix ( double  Rotation_In_Degrees,
char  Axis_Of_Rotation 
)

A rotation matrix translates a vectors such that it would appear to have rotated along Axis_Of_Rotation by an amount equal to Rotation_In_Degrees.

Note that the centre of rotation is the origin and that positive Rotation_In_Degrees will result in clockwise rotation and a negative Rotation_In_Degrees counter-clockwise rotation. See https://en.wikipedia.org/wiki/Rotation_matrix for more info

Parameters
Rotation_In_DegreesThis is how much rotation is required
Axis_Of_RotationThis is which axis of rotation is required - acceptable parameters are x, y and z in char form. If any other char is used, an error message will be displayed on the console
